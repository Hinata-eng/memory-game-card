# 🎮 Memory Game

An educational and fun card-matching game designed to improve memory skills, featuring three difficulty levels:  
- 🟢 **Easy**: Computer hardware components (Laptop, Monitor, CPU, ...).  
- 🟡 **Medium**: Programming concepts (Loop, Algorithm, Variable, ...).  
- 🔴 **Hard**: Networking & Cybersecurity concepts (IP Address, Encryption, Firewall, ...).

---

## 📌 Features
- ⏱️ Timer starts on the first move.  
- 🔄 Reset button to restart the game anytime.  
- 🏆 Winning message showing attempts and time.  
- 🎯 Three difficulty levels.
